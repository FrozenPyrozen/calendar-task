## Instaliation

1. Clone this repository:
```
$ git clone https://github.com/FrozenPyrozen/calendar-task.git
```
2. Change the current directory to **calendar-task**:
```
$ cd ./calendar-task
```
3. Install dependencies using npm/yarn:
```
$ npm install
```
```
$ yarn install
```
4. Run app in terminal:
```
$ npm start
```
